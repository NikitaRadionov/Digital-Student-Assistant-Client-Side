# UC-15: Экспортировать данные платформы

| Атрибут | Значение |
|---|---|
| **Scope** | Цифровой Студенческий Ассистент |
| **Level** | 🌊 Пользовательская цель (sea-level) |
| **Primary Actor** | ЦППРП (Модератор) |
| **Supporting Actors** | Система (формирует CSV, отдаёт файл) |

---

## Цель в контексте

ЦППРП хочет выгрузить актуальные данные о проектах или заявках в CSV — чтобы провести анализ в Excel или передать отчёт руководству.

---

## Стейкхолдеры и интересы

| Стейкхолдер | Интерес |
|---|---|
| ЦППРП | Получить актуальный срез данных для анализа в Excel или отчёта руководству |
| Руководство | Иметь доступ к агрегированной статистике по проектам и заявкам |
| Система | Корректно сформировать CSV с нужной кодировкой; не позволять доступ без соответствующих прав |

---

## Предусловия

1. ЦППРП авторизован с ролью `CPPRP` или `is_staff`

---

## Триггер

ЦППРП нажимает «Скачать CSV» на вкладке экспорта в панели `/cpprp/`.

---

## Гарантии

### Минимальная гарантия
При ошибке (нет прав) файл не отдаётся. Данные в БД не изменяются.

### Гарантия успеха
- Браузер получает HTTP-ответ `Content-Type: text/csv; charset=utf-8` с `Content-Disposition: attachment`
- Файл содержит актуальный срез данных на момент запроса
- Файл корректно открывается в Excel (UTF-8 BOM добавлен)

---

## Основной сценарий успеха

| Шаг | Кто | Действие |
|---|---|---|
| 1 | ЦППРП | Нажимает «Скачать CSV проектов» или «Скачать CSV заявок» |
| 2 | Система | Валидирует права доступа → [BR-MOD-01] |
| 3 | Система | Выполняет запрос к БД (`Project.objects.all()` или `Application.objects.all()`) |
| 4 | Система | Формирует CSV-ответ с UTF-8 BOM; записывает строки потоково |
| 5 | ЦППРП | Получает CSV-файл для скачивания |

---

## Расширения

### 2а. Пользователь не является ЦППРП и не is_staff

- 2а.1. Система возвращает ошибку доступа (HTTP 403)
- **Сценарий завершается**, файл не формируется.

---

## Перечень технологических и пользовательских вариаций

| # | Переменная часть | Варианты |
|---|---|---|
| TV-1 | Тип экспорта | **(A)** `GET /cpprp/export/projects/` → `projects-export.csv` (поля: id, title, status, source_type, team_size, accepted_participants_count, education_program, study_course, created_at); **(B)** `GET /cpprp/export/applications/` → `applications-export.csv` (поля: id, project_id, project_title, applicant_id, applicant_email, status, created_at) |
| TV-2 | Кодировка | UTF-8 с BOM (`﻿`) для совместимости с Microsoft Excel |
| TV-3 | Фильтрация | Отсутствует: всегда полный срез всех записей (`order_by("pk")`) |
