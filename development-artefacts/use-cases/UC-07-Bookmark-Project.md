# UC-07: Сохранить / убрать проект из закладок

| Атрибут | Значение |
|---|---|
| **Scope** | Цифровой Студенческий Ассистент |
| **Level** | 🌊 Пользовательская цель (sea-level) |
| **Primary Actor** | Студент (или любой авторизованный пользователь) |
| **Supporting Actors** | Система (обновляет `favorite_project_ids` в профиле) |
| **Revision** | 1.0 · 2026-05-09 |

---

## Цель в контексте

Пользователь хочет пометить проект как «интересный» или снять пометку — чтобы вернуться к нему позже через вкладку «Закладки» в каталоге.

---

## Стейкхолдеры и интересы

| Стейкхолдер | Интерес |
|---|---|
| Студент | Сохранить заинтересовавший проект и вернуться к нему позже |
| Преподаватель | Его опубликованный проект попадает в закладки — потенциально больше заявок |
| Система | Корректно хранить и обновлять список закладок без дублей; отвечать быстро (нет перезагрузки страницы) |

---

## Предусловия

1. Пользователь авторизован

---

## Триггер

Пользователь нажимает кнопку-закладку (иконку) на карточке проекта или детальной странице.

---

## Гарантии

### Минимальная гарантия
При любой ошибке состояние `favorite_project_ids` не изменяется.

### Гарантия успеха
- Если проект не был в закладках → `pk` добавлен в `profile.favorite_project_ids`; `bookmarked: true`
- Если проект уже был в закладках → `pk` удалён из списка; `bookmarked: false`
- Ответ: `JsonResponse({"bookmarked": bool})`; UI обновляется без перезагрузки страницы

---

## Основной сценарий успеха

| Шаг | Кто | Действие |
|---|---|---|
| 1 | Пользователь | Переключает закладку на карточке проекта или детальной странице |
| 2 | Система | Валидирует существование проекта |
| 3 | Система | Инвертирует состояние закладки: добавляет или удаляет проект из списка сохранённых |
| 4 | Система | Сохраняет обновлённый список закладок в профиле пользователя |
| 5 | Система | Возвращает новое состояние закладки; интерфейс обновляется без перезагрузки страницы |

---

## Расширения

### 1а. Пользователь не авторизован

- 1а.1. Система перенаправляет на страницу входа
- **Сценарий завершается**, список закладок не изменяется.

---

### 2а. Проект не найден (удалён между загрузкой и нажатием)

- 2а.1. `get_object_or_404` возвращает HTTP 404
- **Сценарий завершается**.

## Перечень технологических и пользовательских вариаций

| # | Переменная часть | Варианты |
|---|---|---|
| TV-1 | Хранилище | Список закладок хранится в профиле пользователя как JSON-поле (не отдельная таблица) |
| TV-2 | Транспорт | POST-запрос; ответ — JSON с новым состоянием; клиент обновляет UI без перезагрузки |
| TV-3 | Направление toggle | Идемпотентный: повторное действие всегда инвертирует состояние закладки |
